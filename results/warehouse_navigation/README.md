# Warehouse Navigation Results

Run `matlab/localization_navigation/vlp_wknn_astar_obstacles.m` to generate:

- `01_navigation_path.png`: warehouse layout, positioning result, and A* path
- `02_rss_fingerprint_maps.png`: RSS fingerprint maps for representative LEDs
- `03_positioning_error_cdf.png`: positioning error CDF for different K values
- `04_positioning_error_summary.png`: mean and 90th-percentile positioning errors

MATLAB `.fig` files are generated alongside the PNG files for further editing.
